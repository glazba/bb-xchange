# BB-XChange – Project Context

## Projekt áttekintés

A BB-XChange egy könyv- és társasjáték-csere platform.

A felhasználók:

* regisztrálhatnak és bejelentkezhetnek,
* kezelhetik a profiljukat,
* feltölthetnek könyveket és társasjátékokat,
* böngészhetnek a Marketplace-en,
* több terméket tartalmazó csereajánlatokat küldhetnek,
* elfogadhatnak, elutasíthatnak vagy visszavonhatnak ajánlatokat,
* később üzeneteket és értesítéseket kapnak.

---

# Hibakezelés

## Backend

Minden controller:

```ts
try {
    ...
} catch (error) {
    console.error(error);

    return res.status(500).json({
        message: "Internal server error"
    });
}
```

HTTP státuszok:

* 400 → Bad Request
* 401 → Unauthorized
* 403 → Forbidden
* 404 → Not Found
* 500 → Internal Server Error

---

## Frontend

API:

```ts
if (!response.ok) {
    throw new Error(data.message);
}
```

Komponensek:

```ts
try {
    ...
} catch (error) {
    alert(
        error instanceof Error
            ? error.message
            : "Valami hiba történt."
    );
}
```

Sikeres műveletek után:

```ts
alert("Termék sikeresen létrehozva.");
alert("Termék sikeresen módosítva.");
alert("Ajánlat sikeresen visszavonva.");
```

Minden felhasználói üzenet magyar nyelvű legyen.
